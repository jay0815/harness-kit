## 4. 新开发者阅读路径

### 路径 A: 快速上手 (30 分钟)

1. `tools/base.py` — 理解 Tool 抽象
2. `core/agent_loop.py` — 理解主循环 (关注 `while True` 和迭代逻辑)
3. `core/middleware.py` — 理解中间件 (关注 `AgentMiddleware` 基类和 `RuntimeState`)
4. `agents/sisyphus.py` — 理解入口编排

### 路径 B: 深度理解 (2 小时)

1. `tools/base.py` + `tools/verify_command.py` — 工具系统
2. `core/llm_client.py` — LLM 统一接口
3. `core/agent_loop.py` — 主循环完整流程
4. `core/middleware.py` — 全部中间件
5. `core/error_recovery.py` — 错误恢复
6. `core/memory/layers.py` + `core/memory_middleware.py` — 记忆系统
7. `agents/ralph.py` — 持续执行智能体
8. `agents/team.py` — 并行协作

### 路径 C: 扩展开发 (自定义中间件/工具)

1. 阅读 `core/middleware.py` 中的 `AgentMiddleware` 基类
2. 参考 `LoopDetectionMiddleware` 或 `TodoReminderMiddleware` 作为简单示例
3. 阅读 `tools/base.py` 中的 `Tool` 基类
4. 参考 `tools/verify_command.py` 实现新工具
5. 在 `core/middleware.py` 末尾添加新中间件，在 `main.py` 中注册
