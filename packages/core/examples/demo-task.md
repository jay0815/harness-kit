# Task: Implement a greeting module

Create `src/greet.ts` that exports a `greet(name: string)` function.

Requirements:

- `greet("Alice")` returns `"Hello, Alice!"`
- `greet("")` returns `"Hello, World!"` (empty string defaults to "World")
- `greet("Bob")` returns `"Hello, Bob!"`

After implementing, create `src/greet.test.ts` with unit tests covering all three cases.

Follow the harness-kit workflow phases. Output a `<HK_RESULT>` block after each phase with facts citing the exact text on disk.
